# RedisStorager

### redis-keys:
* **Namespaces**: `ns:{namespace}`
* **Users**: `user:{user_name}`
* **Orgs**: `org:{org_name}`
* **Projects**: `prj:{namespace}:{project_name}`
   * `profile`
   * `labels`
* **Project Revisions**: `rev:{namespace}:{project_name}`
* **Files**: `file:{hash}`
   * `profile`
   * `labels`
   * `download_count`
